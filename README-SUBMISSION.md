## How to Run & Verify

### 1. Install Dependencies
```bash
npm install
npm install --prefix contracts
```

### 2. Run Assessment Tests
To run the automated contract tests:
```bash
npm run test
```

### 3. Run Locally (UI + Local Blockchain)
*   **Terminal 1:** Start the local Hardhat blockchain node:
    ```bash
    npm run contracts:node
    ```
*   **Terminal 2:** Deploy the contracts to the local node:
    ```bash
    npm run contracts:deploy
    ```
*   **Terminal 3:** Start the Next.js frontend dev server:
    ```bash
    npm run dev
    ```
*   **Web Browser:**
    1. Open **http://localhost:3000**
    2. Configure MetaMask (or your preferred Web3 wallet) to connect to Localhost:
       *   **RPC URL:** `http://127.0.0.1:8545`
       *   **Chain ID:** `31337`
    3. Import the private keys of the first two accounts generated by the Hardhat node command in Terminal 1.
    4. Follow the user flow (Create Quest, Accept Quest, Submit Work, Approve & Pay).
